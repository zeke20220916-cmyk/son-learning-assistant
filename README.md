# George的私人小助理

一个面向 iPad 使用的私人学习 PWA。第一版使用纯前端实现，数据保存在当前浏览器的 `localStorage` 中。

## 启动

在项目目录运行：

```powershell
node server.js
```

然后打开：

```text
http://localhost:4173
```

## 当前功能

- 孩子端今日任务
- 完成 / 未完成状态切换
- 自动积分、满环奖励、连续奖励
- 等级称号、正向称号、隐藏称号、负向临时称号
- 家长端周期任务和临时任务设置
- 任务详情手动录入
- AI 任务详情生成的本地模拟入口
- 奖励兑换与记录
- PWA 离线缓存和 iPad 主屏幕安装支持
